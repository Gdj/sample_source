jqScribble

OPTIONS: Dot Black Clear Save Custom Save

Available Options

  • width: Canvas 요소의 너비가 지정되지 않은 경우 부모의 너비가 사용됩니다.
    (기본값 : 300)
  • height: Canvas 요소의 높이를 지정하지 않으면 부모 높이가 사용됩니다.
    (기본값 : 250)
  • backgroundImage: Canvas 배경에 추가 할 이미지입니다.
    (기본값 : false)
  • backgroundImageX: 지정된 배경 이미지를 넣을 캔버스의 X축 오프셋
    (기본값 : 0)
  • backgroundImageY: 지정된 배경 이미지를 넣을 캔버스의 Y축 오프셋
    (기본값 : 0)
  • backgroundColor: 배경을 설정하는 16 진수 색상 값입니다.
    (기본값 : #ffffff)
  • saveMimeType: 이미지가 저장되면 사용되는 유형.
    (기본값 : image/png)
  • saveFunction: Canvas를 저장할 때 사용할 함수.
    (기본값 : BasicCanvasSave)
  • brush: Canvas를 그리는 데 사용할 브러시입니다.
    (기본값 : BasicBrush, [LineBrush, CrossBrush, DotBrush, CircleBrush, SemiCircleBrush, RectangleBrush])
  • brushSize: 사용되는 브러쉬의 크기.
    (기본값 : 2)
  • brushColor: 브러쉬 선의 색상입니다.
    (기본값 : "rgb(0,0,0)" )
  • fillOnClear : 실행시 캔버스가 색상으로 채워지는지 여부를 조정합니다.
    (기본값 : true)
